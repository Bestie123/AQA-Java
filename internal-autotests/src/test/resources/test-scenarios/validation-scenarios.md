# Validation Test Scenarios

## Сценарий 1: Успешный запрос с корректным X-Api-Key
**Тип:** Позитивный
**Приоритет:** Critical

**Предусловия:**
- Приложение запущено на http://localhost:8080
- WireMock запущен на http://localhost:8888

**Шаги:**
1. Отправить POST запрос на /endpoint с заголовком X-Api-Key: qazWSXedc
2. Указать валидный токен и action=LOGIN

**Ожидаемый результат:**
- HTTP статус 200
- result: "OK"

**Постусловия:**
- Нет
